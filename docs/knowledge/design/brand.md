# Brand Direction

The site is a brutalist dark technical lab.

## Visual language

- Near-black background with panel surfaces.
- Acid green accent for signal, focus, active state, and sparse emphasis.
- Warm off-white paper surfaces, graphite blocks, clipped labels, and matte grain can soften the lab feel when the surface needs a more personal builder/founder tone.
- Visible structure: borders, dividers, rails, index-like layouts.
- Monospace UI/body copy with geometric display headings.
- Minimal decoration; ornament should feel like instrumentation, tactile labels, product stickers, or traces.

## Mood

- Public lab notebook.
- Technical, direct, slightly raw.
- Builder/founder, approachable, and practical when the surface is personal-brand first.
- Engineered rather than polished.
- Curious and self-directed.

## Avoid

- Generic portfolio cards with soft shadows and gradients.
- SaaS marketing smoothness.
- Decorative accent overuse.
- Sci-fi dashboards, terminal walls, network diagrams, and fake code as shorthand for "technical."
- Stock personal-brand language.
- Hiding rough process behind fake polish.

## Existing source files

- `src/styles/global.css` contains the global tokens and base styles.
- `docs/design-system.md` documents the current token system.
- `branding-template.html` and `src/pages/og/[...route].ts` should mirror the same visual language.
